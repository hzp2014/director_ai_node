# AI漫导 Android - 快速入门指南

> 快速上手 AI漫导，5分钟制作你的第一部AI漫剧！

---

## 第一步：安装和启动

### 安装应用

1. 下载 APK 文件
2. 在手机上安装
3. 启动应用
4. 授予必要的权限（存储、网络）

### 首次配置

1. 点击右上角 ⚙️ 设置按钮
2. 配置 API Key（推荐使用苍何 API）
   - 智谱 GLM-4.7（智能决策）
   - 视频生成 API（视频合成）
   - 图像生成 API（分镜图片）
3. 保存配置

---

## 第二步：创建第一个剧本

### 输入创意

在首页输入框中输入你的创意：

**示例 1**：
```
制作一个雪地里的冒险故事，主角是一个勇敢的小男孩
```

**示例 2**：
```
樱花树下，女孩在读书的唯美场景，10秒钟
```

**示例 3**：
```
温馨的家庭聚餐场景，三代同堂，快乐氛围
```

### 发送并等待

点击发送按钮 📤，等待 AI 生成剧本（约10-30秒）

---

## 第三步：确认剧本

### 查看剧本内容

进入剧本确认页面，查看：
- 📖 剧本标题和类型
- 👤 角色设定（三视图）
- 🎭 场景列表（旁白和情绪）

### 修改（可选）

如果需要修改：
- 编辑单个场景的提示词
- 在底部输入框输入修改建议
- 点击"重新生成"按钮

### 确认

点击"✓ 确认并继续生成"按钮

---

## 第四步：生成分镜图片

### 自动生成

系统会自动生成分镜图片（约30-60秒/张）

### 查看进度

在剧本卡片上查看进度：
- 🟡 生成中（黄色进度条）
- 🟢 已完成（绿色勾选）

### 单独重试

如果某个场景失败：
- 点击该场景的[重试]按钮
- 重新生成

---

## 第五步：生成场景视频

### 自动生成

所有分镜图片生成完成后，自动开始生成视频（约30-60秒/个）

### 手动触发

如果需要手动开始：
- 在剧本卡片上点击"开始全部生成"按钮

### 播放视频

生成完成后：
- 点击视频播放
- 支持暂停、调节进度

---

## 第六步：合并完整视频

### 合并条件

- ✅ 所有场景视频已生成
- ✅ 没有正在进行的任务

### 合并操作

**方法1：在剧本卡片上**
1. 点击"合并视频"按钮（粉色渐变）
2. 确认合并
3. 等待完成（约1-3分钟）

**方法2：在设置页面**
1. 进入设置
2. 找到"视频合并"卡片
3. 点击"合并场景视频"按钮

### 查看进度

显示合并进度：
- 下载视频 (0-60%)
- 合并处理 (60-95%)
- 清理完成 (95-100%)

---

## 第七步：保存到相册

### 保存单个视频

1. 打开剧本详情
2. 找到要保存的视频
3. 点击"保存到相册"按钮
4. 完成！

### 保存完整视频

1. 在设置页面找到"视频合并"卡片
2. 点击已合并的视频
3. 点击"保存到相册"按钮
4. 完成！

### 保存位置

- 图片：`Pictures/Movies/`
- 视频：`Movies/Movies/`

---

## 常用操作

### 查看历史

点击左上角菜单 (≡) → 查看"历史对话"

### 清理缓存

进入设置 → 缓存管理 → 清理过期缓存

### 导出数据

进入设置 → 数据库查看 → 导出数据 (JSON)

### 切换 API

进入设置 → API 配置 → 点击要修改的服务 → 输入新的 API Key

---

## 快捷技巧

### 💡 提高生成质量

1. **详细描述**: 提供更详细的场景描述
2. **指定风格**: 如"日系动漫风格"、"写实风格"
3. **情绪基调**: 如"温馨"、"紧张"、"唯美"

### 💡 加快生成速度

1. 使用稳定的网络连接
2. 选择高性能的 API 服务
3. 减少场景数量

### 💡 节省存储空间

1. 定期清理缓存（设置 → 缓存管理）
2. 及时保存重要内容到相册
3. 删除不需要的对话

---

## 常见问题速查

| 问题 | 解决方案 |
|------|---------|
| 生成失败 | 检查网络、API Key、余额 |
| 保存失败 | 检查存储权限、空间 |
| 合并失败 | 检查存储空间、清理缓存 |
| 应用崩溃 | 重启应用、清理缓存 |
| 视频卡顿 | 清理后台应用、充电状态 |
